# MIK-1
to jest moja wersja płytki MIK 1, zmodyfikowana nieznacznie w porównaniu do wersji #Zegar, wg projektu Autora
https://klonca80.blogspot.com/2024/04/mik1-czyli-sonda-do-miksid-a.html#more

Zamiast MAX232 zastosowałem gotową płytkę FT 232 RL z gniazdem USB - C, calość zmieściła się w obudowie ELWRO 144 razem z płytka MIK290 /CA80/. Zasilanie MIK 1 ze złącza USB lub z CA80, na PCB są odpowiednie ścieżki do wyboru.
